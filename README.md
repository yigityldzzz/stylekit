# StyleKit

**Extract any website's design system in one click.**

StyleKit is a free Chrome Extension + CLI tool that extracts colors, typography, spacing, border-radius, and shadows from any website — then generates a structured `DESIGN.md` file ready for AI coding assistants.

## How it works

1. Open any website
2. Click the StyleKit extension icon
3. Review extracted design tokens
4. Export as `DESIGN.md`
5. Drop the file in your project root — Claude Code, Cursor, Copilot, or Windsurf instantly understands the design system

## Install

**Chrome Extension** → [stylekit.digitaladexpert.de](https://stylekit.digitaladexpert.de)

**CLI Tool**
```bash
npx stylekit-ai
```

## What gets extracted

| Token | Example |
|---|---|
| Colors | `#635bff`, `#0a2540` |
| Typography | `Inter, 16px, weight 400` |
| Spacing | `8px, 16px, 24px, 32px` |
| Border Radius | `4px, 8px, 12px` |
| Shadows | `0 2px 8px rgba(0,0,0,0.1)` |

## Works with

- **Claude Code** (Anthropic)
- **Cursor AI**
- **GitHub Copilot**
- **Windsurf**
- Any AI tool that reads project files

## Generated DESIGN.md example

```markdown
# DESIGN.md — stripe.com
> Generated by StyleKit

## Colors
primary: #635bff
background: #0a2540
text: #425466

## Typography
font-sans: -apple-system, BlinkMacSystemFont
text-base: 16px
heading-xl: 48px / 700

## Spacing
space-4: 16px
space-8: 32px
space-16: 64px
```
